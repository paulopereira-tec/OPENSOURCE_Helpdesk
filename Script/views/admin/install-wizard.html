Configuração

As seguintes definições são a configuração de toda base da aplicação. Eles são usados ​​para se comunicar com seu banco de dados e definir a conta de um usuário administrador. Por favor, preencha os formulários e clique em Salvar.

Se você já instalou o aplicativo e apenas deseja alterar algumas das configurações, sinta-se livre para atualizar os valores e clique Salvar (Banco de dados existente vai ficar seguro e sem quaisquer alterações).

Seus dados não estão corretos, por favor, verifique as mensagens de erro abaixo e atualize o formulário.

A aplicação não pôde se conectar ao banco de dados fornecido.
Por favor, verifique todos os valores na Configurações de banco de dados seção e certificar-se de seu banco de dados está em execução.

Houve uma falha na tentativa de criar banco de dados e tabelas.
Por favor, verifique todos os valores nas Configurações do banco de dados.

Seu banco de dados - — ainda não existe, crie manualmente usando alguma ferramenta de banco de dados (ex. phpMyAdmin) e voltar depois que é feito.
