Configuração

Bem-vindo a configuração do Chat. Antes de começar a configuração, crie um banco de dados em seu servidor de hospedagem junto a um administrador com todas permissões aprovadas. Você terá de preencher as seguintes informações durante Instalação (se você não sabe esses itens, você pode obtê-los a partir de sua empresa de hospedagem):

1. Local da instalação (geralmente localhost)
2. Porta do servidor (deixe padrão caso não saiba.)
3. Nome do banco
4. Nome do administrador
5. Senha do banco

Se o banco de dados ainda não existe, o assistente de instalação irá tentar criá-lo automaticamente para você. Em alguns servidores, no entanto, essa tentativa irá falhar. Não se preocupe, você poderá criar o banco de dados manualmente usando o manual de instalação e realizar via (phpMyAdmin).
